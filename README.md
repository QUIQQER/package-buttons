QUIQQER CSS Button Klassen
========

Liefert CSS Klassen für a-Links. Erweitert den Inhaltseditor um Button Stile.
Der Benutzer kann somit auf a-Links als Buttons darstellen lassen 

Paketname:

    quiqqer/buttons


Features
--------

- CSS Button Klassen für a-Links
